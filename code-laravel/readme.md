####Install dependencies with `composer`
at root, run command
    
    composer install

####Database
at root/.env, provide database name, username, password for mysql-server

create table for database by command
    
    php artisan migrate

    
####Run app, open browser at localhost:8000
at root, run command
    
    php artisan serve
